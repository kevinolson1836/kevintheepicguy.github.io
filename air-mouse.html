AIR- MOUSE I was browsing reddit and stumbled on a post about a story of a disabled person who was trying to play games on their computer but due to the disabilities they were unable to play the game because they could not use the normal controllers. This person story spoke to me and I knew I needed to do something about it. I used my skill set that I have previous developed to make this thing I call the “AIR-MOUSE”. The main idea is for the person using this to put on the glove and move their hand, the hand movements will directly move the mouse on the screen. All five fingers also have buttons on them that can be mapped to any key on the keyboard. During the construction of this project, I knew I wanted to work with the Arduino Nano 33 BLE. It comes with a 9 axis inertial measurement unit (IMU) which means that it includes an accelerometer, a gyroscope, and a magnetometer. Using the 3 sensors I can track the device in 3d space, using this we can update the mouses position. This also has an accompanying app in python to make this all work. The BLE 33 is supposed to be able to act as a HID (human interface device aka a mouse/keyboard) But at the time of making the glove it was still under development and was not working. So to get past this issue I made the accompanying application. This is not the ideal situation but It works when playing Minecraft. The app connects to the BLE 33 and reads a few values. It will check if we need to move the mouse to the left, right or nothing. This is calculated on the BLE 33 itself and then will send the value over to the Python app. It also checks each button on the fingers and updates the value, The python app will then press the key associated with each finger.
